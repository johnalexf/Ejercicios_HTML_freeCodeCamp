# 🏕️ FreeCodeCamp - Prácticas y Ejercicios

Este repositorio contiene los ejercicios y mini-proyectos desarrollados como parte del plan de estudios de [FreeCodeCamp](https://www.freecodecamp.org/). 

El propósito principal de este espacio es documentar el avance y reforzar las bases de desarrollo Frontend, organizando el código de manera que sirva como evidencia clara del progreso.

## 🚀 Tecnologías Cubiertas

El repositorio está diseñado para escalar por módulos a medida que se avanza en el currículo:

**✅ HTML (Módulo Completado):** Estructura básica, semántica web (nav, main, article), construcción de formularios, tablas estructuradas y aplicación de buenas prácticas de accesibilidad (ARIA).

**⏳ CSS (Próximamente):** Estilos, layouts (Flexbox/Grid) y diseño responsivo.

**⏳ JavaScript (Próximamente):** Lógica de programación, algoritmos básicos y manipulación del DOM.
